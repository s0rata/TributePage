Microsoft Windows

Operating System


Windows Version History (1985-2016):

Windows 1.0

  • Windows 1.0 was released on September 1985. It is not a complete OS; rather, it is extends MS-DOS.

Windows 2.0

  • Windows 2.0 was released in December 1987. It was more popular than its predecessor.
  • Windows 2.1 was released in two different versions. Windows/286 and Windows/386.

Windows 3.0

  • Windows 3.0 was released in 1990. It was the first Windows version to achieve broad commercial success, selling 2 million copies in the first six months.
  • Windows 3.1 was released in March 1992.
  • Windows 3.11 was released in August 1993.
  • Windows 3.2 was released in 1994. It is an updated version of the Chinese version of Windows 3.1.

Windows 9x

  • Windows 95 was released in August 1995. It also introduced a redesigned user interface. It was a major commercial success for Microsoft.
  • Windows NT 3.5 was released in September 1994, focusing on performance improvements.
  • Windows NT 3.51 was released in May 1995, which include additional improvements.
  • Windows NT 4.0 was released in June 1996, introducing the redesigned interface of Windows 95 and NT series.

Windows 2000

  • Windows 2000 was released in February 2000. It was a successor to Windows NT 4.0. Since then, the Windows NT name was dropped.

Windows XP

  • Windows XP was released on October 25, 2011. It introduced a redesigned user interface and also Internet Explorer 6.
  • It has four editions: Home (Consumers), Professional (Business and power users), Media center (Home theater PCs), and Tablet PC (Tablet computers).
  • Windows XP’s server counterpart: Windows Server 2003 was released in April 2003.
  • Windows Server 2003 R2 was released in December 2005.

Windows Vista

  • Windows Vista was released on November 30, 2006, for volume licensing and January 30, 2007 for consumers.
  • Windows Vista’s server counterpart: Windows server 2008 was released in early 2008.

Windows 7

  • Windows 7 and its counterpart, Windows Server 2008 R2 were released as RTM (released to manufacturing), while the former was released to the public 3 months later on October 22, 2009.

Windows 8 and 8.1

  • Windows 8, the successor to Windows 7, was released on October 26, 2012. It introduced to a new Mero user interface with optimizations for touch-based devices.
  • Windows RT was also released devices that utilize the ARM architecture.
  • Windows 8.1 was released on October 17, 2013 as an update to Windows 8.

Windows 10

  • On September 30, 2014, Microsoft announced Windows 10 as the successor to Windows 8.1.
  • Windows 10 was released on July 29, 2015.
  • There are lots of changes to Windows 8.1 such as the return of Start Menu, virtual desktop system, and the ability to run Windows store apps within windows on desktop rather than in full-screen mode.
  • On November 12, 2015, an update to Windows 10, version 1511 was released.
